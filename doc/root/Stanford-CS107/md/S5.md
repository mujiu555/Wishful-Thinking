# Stanford CS 107

## Section V

Example for generic:

``` c
void * lsearch (
  void* key,
  void * base,
  int n,
  int elem_size,
  int (* cmpfn)(void *, void *)
) {
  for (int i = 0; i < n; i ++) {
    void * elemAddr = (u8_t*) base + i * elem_size;
    if (cmpfn (key, elemAddr) == 0) {
      return elemAddr;
    }
  }
  return NULL;
}
```
