## Section 3

### Structure (`struct`)

指针指向结构的起始地址, 其他元素通过相对于起始地址
(基地址,类似汇编的基地址和偏移地址的关系,
汇编的偏移地址以0x10为基,
此处偏移地址以0x1为基且偏移地址的值相等于之前变量的长度的总和)
的偏移访问.

### Array

指针指向数组的起始地址, 其他元素通过相对于起始地址的偏移访问.
总体类似于结构, 但是偏移地址的长度等于n倍的元素变量长度

### Generic

c风格的泛型,

``` c
void swap(void* ap, void* bp, size_t size) {
  byte_t tmp[size];
  memcpy(tmp, a, size);
  memcpy(a, b, size);
  memcpy(b, tmp, size);
}
```
